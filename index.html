HTML for Dummies

What is HTML? HTML (HyperText Markup Language)is a markup language used to create the bones, frame, structure, or scaffold of a website. HTML is mostly seen on front-end development of a website. As noted above, HTML is NOT a programming language but is instead a *markup language*
